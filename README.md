## ğŸš€ Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn
- Appwrite account

### Installation

1. Clone the repository

```bash
git clone https://github.com/Khalid-debugg/FORA.git
```

2. Install dependencies

```bash
cd FORA
npm install
```

3. Set up environment variables

```bash
cp .env.example .env
```

4. Start the development server

```bash
npm run dev
```
